# Test KUHBS definitions

These KUHB definitions are fixtures for the Python tests and installer template.
They intentionally use YAML and Bash script fragments without shebangs.

Examples:

- `test-app/` - AppVM test KUHB
- `test-ndp/` - Named DisposableVM test KUHB
- `test-sta/` - StandaloneVM test KUHB
- `test-udp/` - Unnamed DisposableVM launcher test KUHB
